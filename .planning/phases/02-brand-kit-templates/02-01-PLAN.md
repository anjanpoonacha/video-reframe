## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User file → localStorage | Untrusted file data stored as base64 data URL |
| localStorage → canvas render | Stored data used in drawing operations |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | SVG logo upload | mitigate | Render SVG as bitmap via createImageBitmap in Plan 02; never inject as innerHTML. In this plan: store as data URL only |
| T-02-02 | Denial of Service | Logo file size | mitigate | Enforce 500KB cap in validateLogoFile before FileReader reads file into memory |
| T-02-03 | Tampering | localStorage data | mitigate | Parse with try/catch and merge with typed defaults; never eval stored data |
| T-02-04 | Denial of Service | localStorage quota | mitigate | try/catch around setItem; show user feedback on QuotaExceededError |
| T-02-SC | Tampering | npm install (gsap) | mitigate | Package verified in RESEARCH.md audit — 12yr history, official GreenSock repo, no postinstall scripts |
</threat_model>
